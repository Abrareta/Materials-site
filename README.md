# 📚 Materials Site

A simple web page that provides quick access to course materials organized in Google NotebookLM.

## ✨ Features

- Simple and clean interface.
- Easy subject selection from a dropdown menu.
- One-click access to the selected course notebook.
- Responsive and lightweight design.
- Built using pure HTML, CSS, and JavaScript.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Use

1. Open the website.
2. Select a course from the dropdown menu.
3. Click **"اذهب للرابط 🚀"**.
4. You will be redirected to the corresponding NotebookLM page.

## 📖 Current Courses

- HCM 101
- HCM 113
- PHC 131

## 📌 Purpose

This project was created to make accessing study materials easier by providing a single webpage that links students directly to their course notebooks in NotebookLM.
